# SeleniumLinear
This project is covered one sample test case.

**IDE** : Visual Studio 2019,<br />
**Programming Language** : C#,<br />
**Library Used** : Selenium, Selenium Extras, Nunit.

## How to Clone project:
1. Download and Install Git from https://git-scm.com/downloads,<br />
2. Open Commaand prompt,<br />
3. Enter command git clone https://github.com/stephenshinde/SeleniumLinear.git

## How to Run Project:
1. Go to Visual Studio <br />
2. Click on Build => Clean Solution <br />
3. Click on Build => Clean Solution <br />
4. Click on F5 button
